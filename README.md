# mikomiko 一款集动漫、音乐、galgame、美图于一体的二次元app。

#### 介绍
mikomiko是一款二次元手机app，里面集成了网易云api可以在线听歌。使用scrapy增量式爬虫并整合SpringBoot的完成了的动漫区、美图区、galgame区的api，并应用在app中。实现了在线看番，壁纸欣赏、galgame分享的功能。ps：点击美图区的浮动按钮会触发隐藏美图，"绅士"们不要错过哦。

#### 软件架构
使用AS 4.0开发。
